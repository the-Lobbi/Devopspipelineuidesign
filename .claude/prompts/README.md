# Prompts Library

Reusable prompt templates for common tasks and scenarios.

## Usage

Prompts can be referenced in agent definitions or used directly in conversations.

## Categories

| Category | Description |
|----------|-------------|
| `code/` | Code generation and modification prompts |
| `review/` | Code review and analysis prompts |
| `docs/` | Documentation generation prompts |
| `debug/` | Debugging and troubleshooting prompts |

## Template Variables

Prompts support variable substitution:
- `{{file}}` - Target file path
- `{{language}}` - Programming language
- `{{context}}` - Additional context
- `{{requirements}}` - Specific requirements
